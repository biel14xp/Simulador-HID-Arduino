# 🕹️ Simulador HID - Button Box Customizada

Este projeto foi desenvolvido para a **Avaliação 01 da disciplina de Introdução à TI (CEUB)**. O objetivo é resolver o problema de imersão e agilidade em simuladores de alta performance através de hardware dedicado.

## 🚀 O Problema
Entusiastas de simulação sofrem com a falta de comandos físicos táteis. O uso do teclado aumenta o tempo de resposta e diminui o realismo da experiência.

## 🛠️ Solução Técnica
Desenvolvimento de uma **Button Box** utilizando o microcontrolador **Arduino Leonardo**. 
- **Por que Arduino Leonardo?** Ele possui o chip ATmega32U4, que permite comunicação nativa USB HID (Human Interface Device). O PC reconhece o dispositivo como um Joystick real, sem necessidade de drivers.

## 🧰 Tecnologias e Ferramentas
- **Linguagem:** C++ (Arduino IDE)
- **Hardware:** Arduino Leonardo, chaves de alternância e botões.
- **Prototipagem:** Tinkercad (Circuito) e Figma (Interface).
- **Versionamento:** GitHub.
